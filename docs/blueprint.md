# **App Name**: Gig Strategist

## Core Features:

- Project Brief Input: Provide a chat-like interface for users to paste or type freelance project descriptions, mimicking a conversational AI experience.
- AI Project Plan Generator: An AI tool that analyzes project descriptions, generating a detailed execution plan including necessary steps, potential challenges, and tailored resource suggestions based on the user's registered skills.
- Dynamic Skill-Gap Identification: An AI tool that identifies skills missing or requiring development for a given project, prompts the user to add new skills, and stores this information to enrich future project analyses.
- Interactive Skill Portfolio: A dedicated section for users to manage, add, and update their professional skills, expertise, and certifications, which consistently informs the AI's personalized planning.
- Effort & Complexity Badges: Visually prominent badges displaying an AI-generated recommended time commitment and an overall project difficulty assessment (e.g., beginner, intermediate, advanced) for each project.
- Detailed Solution Blueprint Display: Present an organized breakdown of the project analysis, covering key tasks, suggested tools/technologies, and a structured approach to solving the client's problem, aiding in accurate budgeting and timeline estimations.

## Style Guidelines:

- Primary color: #4B1E78 (A deep, strategic purple, conveying wisdom and expertise, ideal for professional clarity on a light background).
- Background color: #F5F3F8 (A very light, almost white shade with a subtle purple tint, providing a clean and calm canvas for detailed project information).
- Accent color: #4B87F0 (A vibrant, reassuring blue, used to highlight calls to action and important interactive elements, fostering user confidence).
- Headline font: 'PT Sans' (humanist sans-serif) for titles and section headings, offering a blend of modern readability and a touch of warmth.
- Body font: 'Inter' (grotesque-style sans-serif) for all descriptive text and detailed project plans, ensuring maximum legibility and neutrality for complex information.
- Utilize clean, modern line-art icons that visually reinforce concepts of planning, progress, skills, and strategic thinking, ensuring immediate comprehension across the application.
- Implement a clean, modular layout with generous whitespace. Project analysis results should be presented in a distinct, scannable 'chat-bubble' or card-like format, emphasizing clear organized information hierarchy.
- Incorporate subtle, smooth loading animations for AI analysis processes to convey progress. Use soft fade-ins and delicate state changes for interactive elements to provide a polished and responsive user experience.
